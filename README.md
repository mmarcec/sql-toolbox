# SQL Toolbox

A battle-tested SQL Server performance diagnostic toolkit  
for mid–senior developers and DBAs.

Run a full SQL Server health and performance analysis in one command —  
all the scripts you actually use, in one place.

---

## What is SQL Toolbox?

SQL Toolbox is a practical collection of SQL Server diagnostic and performance scripts,
built from real production experience.

It is designed for situations when:
- a database is slow
- blocking or deadlocks appear
- performance issues need answers **now**, not after hours of digging

SQL Toolbox helps you quickly identify **what is hurting your database** and
what actions you should take next.

---

## What problems does it solve?

SQL Toolbox helps you detect:

- 🔍 Blocking sessions and long-running queries
- 💀 Deadlocks (from system_health Extended Events)
- 🐌 Slow and expensive queries
- 📉 Missing indexes with high impact
- ⚙️ Index fragmentation and maintenance candidates
- 📊 Overall database health score
- 🚨 Live performance issues happening *right now*

All without installing agents or external tools.
